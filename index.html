Dodger Color Privacy Policy

NOTE

Dodger Color does not independently collect or share any data on the user other than the user’s unique device identifier. However, Dodger Color utilizes the third party SDK ‘Unity Ads’ which does collect and share all of the specified required data types listed below. Unity Ads does not process any data ephemerally. Unity Ads also encrypts all data in transit and does provide the option to request that the user’s data is deleted.

For more information about requesting data deletion and opting out of sharing visit here.

Dodger Color only stores the following settings and variables attached to the user’s unique device identifier:

  • Highscore
  • Cursor Sensitivity
  • Enemy Speed
  • Maximum Enemy Spawn Rate
  • Minimum Enemy Spawn Rate


DATA COLECTION & SHARING

Under the restrictions of data collection applied to Dodger Color relating to COPPA, GDPR, and others, Unity Ads collects and shares the following

  • User's approximate location
  • User's personal identifiers
  • Data on page views and taps for advertisements in the app
  • App diagnostics
  • Data on the user's device and other identifiers
